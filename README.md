# Used_Cars_Price_Prediction
Using Linear Regression  FRONTEND-Python  Tools Used: Python,Pandas,Numpy,Matplotlib,Seaborn  Library: scikit-learn
Collected and preprocessed a dataset of used cars using Pandas and NumPy, 
handling missing values, duplicates, and categorical variables. Using skewness 
and kurtosis,Outliers Detection by using IQR method 
●Trained and evaluated multiple regression models  
●Evaluated model accuracy using R² score, Mean Absolute Error (MAE), and 
Root Mean Squared Error (RMSE).
